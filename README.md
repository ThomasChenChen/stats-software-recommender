# stats-software-recommender

A single-page tool that helps psychology students choose the right statistical software for their workflow.

## Usage

Open `preview.html` in any modern web browser, or visit the live demo via GitHub Pages.

Answer the six questions and click **Get recommendations** to see ranked suggestions with reasoning.

## Files

- `preview.html` – the full application (self-contained, no dependencies)
- `index.html` – redirects to `preview.html` for GitHub Pages root URL support
